How to connect Uplisting data

  1. Request Uplisting API access at support@uplisting.io
  2. Get your API key at app.uplisting.io/connect/api
  3. Open Google Apps Script, create a new project and paste the contents of apps-script.gs
  4. In the script editor go to Project Settings → Script Properties and add key UPLISTING_KEY with your API key
  5. Click Deploy → New deployment → Web App (Execute as: Me · Who has access: Anyone)
  6. Copy the Web App URL and paste it into the field above
